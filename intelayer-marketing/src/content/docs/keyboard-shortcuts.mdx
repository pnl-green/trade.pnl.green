# Keyboard Shortcuts

Intelayer prioritizes speed. The following shortcuts apply across the desktop terminal. Press `?` in-app for a searchable overlay.

## Global navigation

- `⌘/Ctrl + K` – Command palette
- `⌘/Ctrl + Shift + M` – Switch workspaces
- `⌘/Ctrl + P` – Toggle portfolio panel
- `⌘/Ctrl + Shift + S` – Save current layout

## Trading actions

- `T` – Toggle AI-CLI overlay
- `B` – Open order ticket
- `L` – Flatten active position
- `Shift + Enter` – Submit current order

## Risk & automation

- `⌘/Ctrl + R` – Open risk manager
- `⌘/Ctrl + B` – Launch bot builder
- `⌘/Ctrl + Shift + A` – Manage alerts

Keyboard shortcuts are customizable under **Settings → Input**. Profiles sync locally per device, so experimental bindings won’t disrupt teammates.
